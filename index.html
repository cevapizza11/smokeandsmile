Configuration Firebase

Pour synchroniser sur tous tes appareils, connecte ta base Firebase gratuite.

Va sur console.firebase.google.com → Créer un projet → nomme-le smoke-smile

Firestore Database → Créer une base → Mode Production → Région europe-west3 (Frankfurt)

Règles Firestore → remplace tout par : rules_version='2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } } → Publier

⚙️ Paramètres du projet → Tes applications → Ajouter une appli Web → Copie et colle l'objet firebaseConfig ci-dessous
